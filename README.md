# Ticket System (Flask + SQLite)

A simple, clean, and professional IT ticketing system built to understand how real help desk
ticket workflows operate. This project models the core features of tools like ServiceNow,
JIRA Service Desk, and Zendesk.

## Features
- Create tickets with priority, category, and description
- Assign tickets to technicians
- Update ticket status (Open, In Progress, Closed)
- Add notes to tickets
- Filter tickets by priority and status
- SQLite database (no setup required)
- Clean Bootstrap UI

## Tech Stack
- Python / Flask
- SQLite
- SQLAlchemy
- Bootstrap
